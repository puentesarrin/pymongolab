mongo_client – Connecting to MongoLab¶

class pymongolab.mongo_client.MongoClient(api_key, version='v1', proxy_url=None)[source]¶

Bases: object

Instance class with the API key located at https://mongolab.com/user?username=[username].

Example usage:

>>> from pymongolab import MongoClient
>>> MongoClient("MongoLabAPIKey")
MongoClient('MongoLabAPIKey', 'v1')

Note

The version parameter is optional, because it is planed for using in future versions of REST API.

When your connection needs to set a proxy, you can to set an str with the Proxy url to proxy_url parameter. If you don’t set a proxy_url, then mongolabclient.client.MongoLabClient gets system proxy settings.

>>> from pymongolab import MongoClient
>>> MongoClient("MongoLabAPIKey", proxy_url="https://127.0.0.1:8000")
MongoClient('MongoLabAPIKey', 'v1')
