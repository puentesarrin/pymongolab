PyMongoLab¶

PyMongo-flavored package for accessing to MongoLab databases via MongoLabClient.

pymongolab.ALL = 2¶

Profile all operations.

pymongolab.ASCENDING = 1¶

Ascending sort order.

pymongolab.DESCENDING = -1¶

Descending sort order.

pymongolab.OFF = 0¶

No database profiling.

pymongolab.SLOW_ONLY = 1¶

Only profile slow operations.

Sub-modules:

  • mongo_client – Connecting to MongoLab
  • connection – Connecting to MongoLab – DEPRECATED
  • database – Database level operations
  • collection – Collection level utilities for MongoLab
  • cursor – Cursor class to iterate over MongoLab query results
