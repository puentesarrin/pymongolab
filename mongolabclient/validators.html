validators – Tools for checking operations parameters¶

mongolabclient.validators.check_api_key(api_key)[source]¶

Check if API key match with the following regular expression: [a-z0-9]{24}.

mongolabclient.validators.check_database_name(name)[source]¶

Check if database name contains invalid characteres.

mongolabclient.validators.check_document_to_update(document)[source]¶

Check if the update operators from a document are valid.

mongolabclient.validators.check_documents_to_insert(doc_or_docs)[source]¶

Check if dict item type is an instance of list or an instance of dict.

mongolabclient.validators.check_list_documents_params(**kwargs)[source]¶

Check parameters for REST API list-documents operation evaluating if they are correct, removing them if no have a value or raise a TypeError if they have a not expected type.

mongolabclient.validators.remove_empty_params(params)[source]¶

Remove items from a dict if no have a value.
