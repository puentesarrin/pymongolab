class mongolabclient.settings.MongoLabSettings(version='v1')[source]¶

Bases: object

Instance class to get url and operations detail depending the version REST API.

All of this data are specified on MongoLab Support website: http://support.mongolab.com/entries/20433053-rest-api-for-mongodb

Note

The version parameter is optional, because it is planed for using in future versions of REST API.
