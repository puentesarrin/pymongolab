client – HTTP client for MongoLab REST API¶

class mongolabclient.client.MongoLabClient(api_key, version='v1', proxy_url=None)[source]¶

Bases: object

Instance class with the API key located at https://mongolab.com/user?username=[username].

Note

The version parameter is optional, because it is planed for using in future versions of REST API.

When your connection needs to set a proxy, you can to set an str with the Proxy url to proxy_url parameter. If you don’t set a proxy_url, then MongoLabClient gets system proxy settings.

>>> from mongolabclient import MongoLabClient
>>> MongoLabClient("MongoLabAPIKey", proxy_url="https://127.0.0.1:8000")
MongoLabClient('MongoLabAPIKey', 'v1')
